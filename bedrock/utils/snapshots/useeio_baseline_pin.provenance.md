# USEEIO baseline pin - provenance

The companion file **`useeio_baseline_pin.json`** pins a specific **`.xlsx`** object in **GCS** (`gs_uri` + `sha256`) so diagnostics can reproduce the same bytes. That workbook is a **frozen export** used for benchmarking; it is not generated by Bedrock.

## USEEIO Model lineage

The pinned artifact is named **`USEEIOv2.6.0-phoebe-23`** (see `model_version_label` in the JSON). That follows EPA’s **USEEIO** naming (version, **Phoebe** model alias, economic reference year). **Phoebe** denotes a particular USEEIO model configuration in EPA's registry; see the upstream model list:

https://github.com/cornerstone-data/USEEIO/blob/master/models.md
